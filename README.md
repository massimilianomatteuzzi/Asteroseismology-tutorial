# Asteroseismology-tutorial
Tutorial for the PhD course in asteroseismology at the University of Bologna.
For the lesson, one needa a working ```python3``` envirnoment with ```jupyter-notebook``` installed (https://jupyter.org/install)

Please, before using the ```jupyter-notebook``` locally, be sure to have ```scipy```, ```plotly```, ```IPython```, ```numpy```, ```lightkurve``` and ```matplotlib``` installed in a working ```python3``` environment.
